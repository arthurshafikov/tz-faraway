# Faraway TZ Server

Сервер для демонстрации работы с библиотекой

# Commands 

## Docker-compose

Run the application
```
make up
```

Down the application
```
make down
```

## Tests

Run the unit tests
```
make test
```

Run linters
```
make lint
```

## App

Run the server without docker
```
make run
```
