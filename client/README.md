# Faraway TZ Client

Клиент для демонстрации работы с библиотекой

# Commands 

## Docker-compose

Run the application
```
make up
```

Down the application
```
make down
```

## Tests

Run the unit tests
```
make test
```

Run linters
```
make lint
```

## App

Run the client without docker
```
make run
```
